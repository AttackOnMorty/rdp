# RDP

RDP stands for **Recursive Descent Parser**. The best way to learn from this repo is through the test cases.

## Run Tests

```
yarn install && yarn test
```

## Run ClI

```bash
# Run expression
./bin/rdp -e 'let x = 1;'
# Run file
./bin/rdp -f ./bin/example.txt
```
